Get pods in a namespace

kubectl get pods -n <name_space>

Restart a service

kubectl rollout restart deployment <service> -n <name_space>

Get logs from pod

kubectl logs <pod_name>  -n <namespace> <container_name>

List all namespaces in a kubernetes cluster

kubectl get ns

List all resources in a namespace

kubectl get all -n <name_space>
